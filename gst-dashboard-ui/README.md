# GST Dashboard UI Patch

Replace your existing frontend `src/` folder with the files in this package.

Install the required dependencies:

```bash
npm install react-router-dom axios recharts framer-motion lucide-react
```

If needed, set the backend base URL:

```bash
REACT_APP_API_BASE_URL=http://127.0.0.1:8000
```

Then run:

```bash
npm start
```
